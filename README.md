# 🔑 Random Password Generator

A sleek, responsive web application that generates secure, random passwords based on user input. This project demonstrates core front-end development skills, focusing on DOM manipulation and algorithmic logic.

## 🚀 Live Demo
[View the Live Project](https://mooraxx1.github.io/password-generator/)

## 🛠️ Features
* **Custom Length:** Allows users to define the length of the generated password.
* **Instant Copy:** Features a one-click copy-to-clipboard function for better UX.
* **Fully Responsive:** Optimized for desktop, tablet, and mobile viewing.
* **Clean UI:** Minimalist design focused on usability.

## 💻 Tech Stack
* **HTML5:** Semantic structure for accessibility.
* **CSS3:** Custom styling and responsive layouts.
* **JavaScript (Vanilla):** Logic for random string generation and clipboard API integration.
